# Atena's Personal Website 🌸

A personal GitHub Pages website for Atena — a compassionate nurse with a purple & pink theme.

## File Structure

```
atena/
├── index.html          ← Home page
├── about.html          ← About Me
├── career.html         ← Nursing Career
├── academic.html       ← Academic Journey
├── lifestyle.html      ← Hobbies & Lifestyle
├── gallery.html        ← Photo Gallery
├── contact.html        ← Contact page
├── css/
│   └── style.css       ← All styles (purple-pink theme)
├── js/
│   └── main.js         ← Navigation & animations
├── images/
│   ├── README.txt      ← Instructions for adding photos
│   ├── profile.jpg     ← YOUR profile photo (add this!)
│   └── gallery1-9.jpg  ← Gallery photos (add these!)
└── README.md           ← This file
```

## How to Personalise

1. **Add your photos** → Place them in the `images/` folder (see `images/README.txt`)
2. **Update your email** → Search for `your@email.com` across all HTML files and replace
3. **Update Instagram** → Search for `yourusername` and replace with your handle
4. **Update content** → Edit any text in the HTML files to match your real details
